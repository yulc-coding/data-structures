# 栈
## 只允许访问一个数据项，即最后插入的数据项，叫做栈的顶(top)
## 属于后进先出（LIFO）
## 通常很小，时临时的数据机构
## push（进栈）
## pop（出栈）
## top 指向栈顶的数据；
## 可以通过数组或者链表的方式实现

## 例子
* java中的方法调用用到了入栈、出栈
* 文本符号匹配 如 '(' 匹配 ')'，'{' 匹配 '}'，'[' 匹配 ']'
