# 堆

### 特性
* 完全二叉树：除了树最后一层节点不需要时满的，其他每一层的节点从左到右都是满的。
* 最小堆：每一个节点都小于它的子节点
* 最大堆：每一个节点都大于它的子节点
* 无法按顺序遍历节点

### 删除最大节点（向下筛选）
1. 移除根节点；
2. 把**最后一个节点**（最后一层最右边的节点）移到根的位置；
3. 向下筛选，和**子节点中大的节点**交换位置；
4. 重复上述步骤，直到遇到子节点都小于它或者没有子节点为止；

### 插入节点（向上筛选）
1. 将节点插入当前堆的最后一个位置；
2. 如果大于它当前的父节点，向上筛选，和父节点交换位置；
3. 重复上述步骤，直到小于父节点或者处于根节点为止；
